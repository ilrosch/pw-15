Реализовать основные операции бинарного дерева поиска (BST)

Класс TreeNode для узла дерева

Класс BinarySearchTree с методами:

- insert(key, value) - вставка элемента

- search(key) - поиск элемента

- delete(key) - удаление элемента

- height() - высота дерева

- is_balanced() - проверка сбалансированности

Технические требования
- Python 3.8+
- UV для управления зависимостями
- Инструкция для запуска и проверки методов

# Run
```
    uv run bts.py
```
